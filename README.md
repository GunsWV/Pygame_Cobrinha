# 🐍 Jogo da Cobrinha (Snake Game)

Projeto simples desenvolvido em Python utilizando a biblioteca Pygame.

## 🎮 Sobre o projeto

Este é o clássico jogo da cobrinha, onde o jogador controla uma cobra que deve comer frutas para crescer e aumentar sua pontuação.

O jogo termina quando:

* A cobra bate nas paredes
* A cobra colide com o próprio corpo

## 🕹️ Controles

* ⬆️ Seta para cima → mover para cima
* ⬇️ Seta para baixo → mover para baixo
* ⬅️ Seta para esquerda → mover para esquerda
* ➡️ Seta para direita → mover para direita

## ⚙️ Tecnologias utilizadas

* Python
* Pygame

## ▶️ Como executar

1. Instale o Python
2. Instale o Pygame:

```
python -m pip install pygame
```

3. Execute o arquivo:

```
python nome_do_arquivo.py
```

## 🧠 Funcionalidades

* Movimentação da cobra
* Sistema de pontuação
* Crescimento da cobra ao comer a fruta
* Colisão com paredes
* Colisão com o próprio corpo
* Tela de Game Over
